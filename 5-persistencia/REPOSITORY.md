Repository
Pode ser utilizado com o Active Record
Manipular objetos em grupo
Operações em lote
Exemplos:

- getPessoasPorNome($nome)
- getPessoasPorCidade($cidade)
- deletePessoasPorCidade($cidade)

Um Repository é uma camada que atua como um gerenciador de coleções
Active Record "invidual, by ID"
Repository "em grupo, vários filtros"

Filtros de consulta ou exclusão de objetos..
